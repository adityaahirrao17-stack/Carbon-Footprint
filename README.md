# Zerofy - AI-Powered Personal Climate Action Platform

A modern, comprehensive climate action platform based on SDG 13 (Climate Action), designed using Design Thinking principles to empower individuals to make informed climate decisions.

## 🌍 Features

### Landing Page
- **Hero Section**: Engaging introduction with animated carbon counter
- **Problem Definition**: Visual cards explaining climate challenges (Rising Heat, Transport Emissions, Lifestyle Impact)
- **Solution Overview**: Three-pillar approach to climate action

### Dashboard
- **Carbon Score**: Personal climate grade with visual indicators
- **Emission Breakdown**: Interactive donut chart showing carbon footprint by category
- **Monthly Trend Analysis**: 5-month emission reduction tracking
- **30-Day Climate Missions**: Habit tracking with streak counters
- **Personalized Roadmap**: Timeline of suggested climate actions

### AI Climate Decision Assistant
- **Interactive Chat Interface**: Ask questions about daily climate decisions
- **Smart Recommendations**: AI-powered travel mode comparison
- **Weather-Aware Suggestions**: Context-sensitive advice
- **Impact Projections**: See long-term effects of your choices

### Community Impact Simulator
- **Interactive Sliders**: Adjust user count and behavior patterns
- **Real-Time Calculations**: Dynamic CO₂ savings, trees equivalent, cars off road
- **6-Month Projections**: Visual charts showing cumulative impact
- **Action Comparison**: Bar charts comparing different climate actions

### City Climate Initiatives
- **Initiative Grid**: Browse government and community programs
- **Category Filtering**: Tree plantation, renewable energy, transport, waste management
- **Participation Tracking**: See active members and impact metrics
- **Volunteer Sign-up**: Easy engagement with local projects

## 🎨 Design System

### Color Palette
- **Primary Green**: `#1B5E20` - Trust, nature, action
- **Secondary Teal**: `#26A69A` - Progress, innovation
- **Accent Orange**: `#FF7043` - Urgency, energy
- **Background Mint**: `#F1F8F6` - Clean, breathable
- **Success Green**: `#43A047` - Achievement, growth

### Typography
- **Headings**: Poppins (Bold/SemiBold)
- **Body Text**: Inter (Regular/Medium)

### Layout Principles
- 8px spacing grid system
- 16px rounded corners
- Soft shadows (low elevation)
- Minimal outline icons
- Clear visual hierarchy
- Fully responsive design

## 💡 Design Philosophy

Built using Human-Centered Design Thinking:

1. **Empathy**: Makes users feel understood, not judged
2. **Define**: Shows climate problems in relatable terms
3. **Ideate**: Presents simple, actionable solutions
4. **Prototype**: Clean dashboard-based interaction
5. **Test**: Clear hierarchy, intuitive navigation
